2. **Select**:

   - Report Type (Weekly, Monthly, Quarterly)
3. **Click**: "Generate Professional Report"
4. **See all 9 sections** appear:
   - Cover page
   - Performance overview
   - Content leaderboard
   - Audience insights
   - Growth charts
   - Insights & recommendations
   - Action plan
5. **Download**: PDF with all sections
