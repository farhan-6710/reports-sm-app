## 🚀 How to Create Your First Report

### Step 1: Open Dashboard
Go to: **http://localhost:3000**

### Step 2: Click "Create Report"
Look for the large white button in the top navigation bar

### Step 3: Follow the 4-Step Wizard

#### 📱 Step 1: Platform & Account
- Choose platform (Instagram or Facebook)
- Select report period (Last 7 Days or Last 30 Days)
- Enter account name (e.g., @mybrand)
- Click **"Next"**

#### 👥 Step 2: Audience Metrics
Fill in:
- **Total Followers**: Current follower count
- **Following**: Accounts you follow
- **Posts Published**: Posts in selected period
- **Total Impressions**: Times content was shown
- **Accounts Reached**: Unique accounts reached
- **Profile Visits**: Profile page views

Click **"Next"**

#### 💬 Step 3: Engagement Data
Fill in:
- **Total Likes**: All likes on posts
- **Total Comments**: Comments received
- **Shares**: Times content was shared
- **Saves**: Times content was saved
- **Website Clicks**: Link clicks
- **Email/Contact Clicks**: Contact button taps

**For Instagram:**
- **Story Views**: Story impressions
- **Story Replies**: DMs from stories
- **Video Views**: Total video plays
- **Reel Plays**: Reel views

Click **"Next"**

#### ✅ Step 4: Review & Generate
- See **auto-calculated metrics**:
  - Total Engagement
  - Engagement Rate
  - Average per Post
- Review summary of all data
- Check completion percentage
- Click **"Generate Report"** 📊

## 📱 Where to Find Instagram Metrics

### Open Instagram App:

1. **Go to Profile** → Tap three lines (☰)
2. **Tap "Insights"**
3. **Select time period** (7 or 30 days)

### Metrics Location:

**Overview Tab:**
- Accounts Reached → Reach
- Accounts Engaged → (calculate from interactions)
- Total Followers → Followers
- Content Interactions → Likes + Comments + Shares + Saves

**Content Tab:**
- Posts you shared → Posts count
- Impressions → Total impressions
- Profile visits → Profile Views

**Activity Tab:**
- Website taps → Website Clicks
- Email button taps → Email Clicks
- Profile activity → Profile visits

**Stories:**
- Story interactions → Story views
- Story replies → Story replies

---

## 📘 Where to Find Facebook Page Metrics

### Open Facebook Page:

1. **Go to Your Page**
2. **Click "Insights"** (left menu)
3. **Select time period**

### Metrics Location:

**Overview:**
- Page Likes → Followers
- Page Reach → Reach
- Page Impressions → Impressions
- Page Views → Profile Views

**Posts:**
- Post Reach → Reach
- Post Engagement → Likes + Comments + Shares
- Post Clicks → Website Clicks

**People:**
- Page Followers → Total Followers
- Page Views → Profile Visits

## 📊 Auto-Calculated Metrics

The system automatically calculates:

### 1. Total Engagement
```
Likes + Comments + Shares + Saves
```

### 2. Engagement Rate
```
(Total Engagement / Followers) × 100
```

### 3. Average Engagement per Post
```
Total Engagement / Posts Count
```
